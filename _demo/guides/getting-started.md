Welcome to your Obsidian-powered Jekyll site.

## Quick Start

1. **Create notes** — Add `.md` files to the `_notes/` directory
2. **Link notes** — Use `[[wikilinks]]` to connect ideas
3. **Add tags** — Use `#hashtags` in your content
4. **Build** — Run `bundle exec jekyll serve`

## Features

- **Wikilinks** — `[[note-name]]` syntax
- **Backlinks** — Automatically generated
- **Graph view** — Visualize connections
- **Search** — `Ctrl/Cmd + K` to search

> [!tip]
> Start by creating a few notes and linking them together to see the graph come alive.
